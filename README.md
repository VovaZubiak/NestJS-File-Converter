# Universal Media Converter

## Description

Веб-додаток для безпечної конвертації медіафайлів та документів.
**Стек:** NestJS (Backend) + Python (Worker) + FFmpeg.

## Можливості
* **Зображення:** JPG, PNG, WEBP, GIF, PDF
* **Відео/Аудіо:** MP4, AVI, MOV, MP3, WAV
* **Документи:** DOCX → PDF, TXT → PDF, PDF → TXT

## Попередні вимоги
* **Node.js** (v25+)
* **Python** (v3.0+)
* **Git**

## Установка

**1. Клонування та залежності**
```bash
git clone [https://github.com/VovaZubiak/NestJS-File-Converter.git](https://github.com/VovaZubiak/NestJS-File-Converter.git)
cd converter-backend

npm install

pip install Pillow python-docx fpdf PyPDF2

## Project setup

$ npm install
npm run start:dev
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
